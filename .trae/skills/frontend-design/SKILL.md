# Frontend Design Skill

This skill helps create modern, futuristic, and clean user interfaces with advanced design aesthetics.

## Design Principles

1. **Futuristic Aesthetic**
   - Glassmorphism effects with backdrop blur
   - Neon accent colors with subtle glow effects
   - Smooth animations and micro-interactions
   - Dark/Light theme support with seamless transitions

2. **Clean & Modern**
   - Minimalist design with ample whitespace
   - Clear information hierarchy
   - Consistent color palette and typography
   - Card-based layouts with subtle shadows

3. **User Experience**
   - Intuitive navigation and interactions
   - Responsive design for all screen sizes
   - Accessibility considerations
   - Loading states and error handling

## Color Palette

**Primary Colors:**
- Blue: #3b82f6 (Sky Blue)
- Purple: #8b5cf6 (Violet)
- Cyan: #06b6d4 (Cyan)

**Accent Colors:**
- Neon Blue: #00d4ff (for highlights)
- Neon Purple: #a855f7 (for emphasis)
- Success: #10b981
- Warning: #f59e0b
- Error: #ef4444

**Backgrounds:**
- Dark: #0f172a (Slate 900)
- Dark Secondary: #1e293b (Slate 800)
- Light: #f8fafc (Slate 50)
- Light Secondary: #e2e8f0 (Slate 200)

## Typography

- **Primary Font:** Inter or system-ui
- **Headings:** Bold, 1.25-2x scaling
- **Body:** Regular weight, 16px base
- **Monospace:** Fira Code or JetBrains Mono for code

## Component Design Guidelines

### Buttons
- Primary: Gradient backgrounds with hover glow
- Secondary: Outline or ghost styles
- Icon buttons: 40x40px, rounded full

### Cards
- Glassmorphism: rgba(255,255,255,0.1) with backdrop blur
- Subtle borders: 1px solid rgba(255,255,255,0.2)
- Rounded corners: 12-16px
- Hover elevation: subtle shadow increase

### Inputs
- Floating labels with animations
- Focus states with neon glow
- Clear validation states
- Icon integrations

### Navigation
- Sidebar with smooth transitions
- Active states with neon indicators
- Collapsible/expandable sections
- Breadcrumb navigation

### Charts & Data Visualization
- ECharts with custom theme
- Gradient fills and neon accents
- Smooth animations on data change
- Responsive chart sizing

## Animations

- **Entrance:** Fade + slide up
- **Exit:** Fade + slide down
- **Hover:** Subtle scale + glow
- **Loading:** Skeleton shimmer or spinner
- **Scroll:** Reveal animations

## Layout Patterns

1. **Dashboard Layout**
   - Fixed sidebar (collapsible)
   - Top header with user profile
   - Main content area with grid
   - Responsive grid cards

2. **List/Table Layout**
   - Search and filter bar
   - Data table with pagination
   - Bulk actions
   - Row-level operations

3. **Form Layout**
   - Multi-step wizard or single page
   - Field grouping with sections
   - Real-time validation
   - Progress indicators

4. **Detail View Layout**
   - Tabs or accordion sections
   - Sidebar with quick actions
   - Timeline/activity feed
   - Related items section

## Usage

Invoke this skill when:
- Designing a new frontend application
- Creating UI components or pages
- Refreshing an existing interface with modern design
- Implementing dark/light theme switching
- Creating dashboard or admin interfaces
